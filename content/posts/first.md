Okay, so I struggled with this for a good few hours, but I managed to set up the site from scratch using Hugo. The best part is the synchronization with GitHub and the ability to update content directly from the coding app. So, instead of typing:

- git add .

- git commit -m "Adding a new entry or whatever"

- git push

you can simply use VSCode for this.
